# Typography

Spirit utilizes a typgoraphy system to create a pleasent and consistent visual indentity and hierarchy of content.

### Font

Spirit utilizes the Inter font family as the primary font.

### Scale

The minor third scale is used to help determine the font size that should be applied to help keep a consistent scale as font sizes changes between different text elements.
